AI 拟人化驱动控制台 (2025版)
本项目是一个基于 Interception 驱动 的 Windows 自动化控制工具，集成了驱动级按键模拟、贝塞尔拟人鼠标轨迹、窗口子区域精准绑定以及智能疲劳模拟系统。
核心功能
驱动级模拟：使用内核驱动绕过应用层检测，支持 RDP 远程桌面及各类窗口。
子窗口绑定：精准锁定目标区域（如记事本编辑区、RDP 渲染区），自动获取分辨率。
拟人鼠标轨迹：基于贝塞尔曲线算法，模拟真实手部移动。
智能疲劳系统：随机触发疲劳停顿，模拟真人操作习惯。
可视化操作：提供 GUI 界面，支持拖拽探测窗口句柄。
安装指南
1. 安装 Python 环境
推荐使用最新的 Python 3.14+。
前往 Python 官网 下载并安装。
注意：安装时务必勾选 "Add Python to PATH"。
2. 安装 uv 依赖管理工具
uv 是目前最快的 Python 包管理工具。打开终端（PowerShell），运行以下命令：
powershell
powershell -c "irm astral.sh | iex"
请谨慎使用此类代码。

3. 安装 Interception 驱动 (必须)
由于本项目涉及底层驱动模拟，必须安装 Interception 驱动：
下载驱动安装包：Interception GitHub Releases。
解压后，以管理员身份打开 PowerShell。
进入 command line installer 目录。
运行安装命令：.\install-interception.exe /install。
重启电脑（不重启驱动无法生效）。
4. 初始化项目与依赖
在项目根目录下运行：
powershell
# 初始化环境
uv venv
# 激活环境 (Windows)
.\.venv\Scripts\activate
# 安装核心依赖
uv add interception-python pyclick pywin32 pynput
请谨慎使用此类代码。

使用说明
1. 启动程序
必须以 管理员权限 运行终端或 IDE，否则驱动和置顶窗口功能将失效。
powershell
uv run main.py
请谨慎使用此类代码。

2. 绑定窗口
程序启动后，点击并按住界面上的蓝色按钮 “● 拖拽准星到目标子区域”。
将鼠标移动到目标位置（如记事本的白色打字区）并松开。
界面会显示当前绑定的 HWND（句柄）、类名 及 分辨率。
3. 快捷键控制
按键	功能描述
7, 8, 9, 0	执行预设的驱动级按键序列（如上跳、闪避等）
1	执行“拟人乱晃”：先移至中心，再按住左键画乱线
F8	暂停 / 恢复 当前所有正在执行的任务队列
F9	清空 当前积压的任务队列
ESC	立即停止脚本并安全退出
注意事项
前台焦点：由于使用的是驱动级硬件模拟，执行操作时程序会自动切换焦点至目标窗口。
RDP 使用：操作远程桌面时，请确保 RDP 窗口处于显示状态，不要最小化。
安全性：本项目包含 20-50 秒随机触发的 1-3 秒“疲劳停顿”，旨在提高拟人度。
